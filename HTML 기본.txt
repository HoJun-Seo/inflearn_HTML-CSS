- 태그 란?
HTML 문서(웹 서비스를 하기 위한 기본적인 문서)를 구성하고 있는 요소, 태그에는 이름과 속성이 있다.
/ 가 붙어있는 태그 : 열린 태그(시작 태그)
/ 가 붙어있지 않은 태그 : 닫힌 태그(종료 태그)
시작 태그와 종료 태그는 항상 쌍을 이룬다.
<img src = "이미지 명"> - img : 태그 이름, src : 태그 속성, 이미지 명 : 속성 값
HTML 버전은 HTML5 를 활용한다.
HTML5 는 어떤 웹 사이트를 서로 다른 브라우저에서 볼 때 약간의 차이가 존재하는 것을 막기 위해 웹 표준에 대한 부분을 많이 지키고 있다.

<!DOCTYPE html> <!--문서의 타입을 지정(html 파일)-->

<head><!--문서의 정보 및 타이틀을 지정하는 태그-->
    <!--스타일 시트의 내용, 자바스크립트의 내용과 같은 환경설정에 관한 정보들이 존재할 수 있다.-->

<body>
    <!--전달하고자 하는 정보들이 들어가는 태그-->

<br /> <!--개행 태그, 종료태그(/)와 시작태그(br)가 한 몸으로 되어 있다.-->

<h1>태그입니다.</h1><!--페이지의 타이틀이 될 태그, 숫자가 커지면 커질수록 글자의 크기가 작아진다, 최대 6까지 존재한다.-->
    <h2>태그입니다.</h2><!--개행 태그를 따로 넣어주지 않아도 한 단락을 이루고 있기 때문에 바로 개행이 되어진다.-->
    <h3>태그입니다.</h3>
    <h4>태그입니다.</h4>
    <h5>태그입니다.</h5>
    <h6>태그입니다.</h6>

<p>html 학습용 <br />p 태그</p><!--본문에 사용되는 태그, 하나의 문단이자 단락이 되는 태그이다.-->

<p><a href="http://www.google.com">구글 홈페이지</a></p><!--a 태그 : 다른 웹 사이트 하이퍼 링크 태그, href 속성에 어떤 사이트로 갈지 경로를 지정해준다.(프로토콜이 있다면 프로토콜 까지 정확하게 기입할 것) -->
