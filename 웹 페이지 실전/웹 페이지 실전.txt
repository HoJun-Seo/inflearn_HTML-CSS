http://www.yahoo.com 을 통해서 전체적인 레이아웃을 만들어 보자.
위와 같은 특정 홈페이지에서 우클릭 -> 페이지 소스 보기를 통해 해당 사이트의 소스 코드를 보면서 
전체는 아니더라도 내가 원하는 곳의 소스 코드가 어떻게 되어 있는지 보는 것 만으로도 공부가 될 수 있다.
웹 페이지 실전 - 1 강의 핵심
: 수직 구조인 li 태그를 수평구조로 전환해 주는 것 (overflow : hidden, float : left)
  div 태그 또한 수평적으로 정렬 가능하다.
  해당 강의에서 사용된 소스 코드를 다운 받아서 직접 분석해보자.

웹 페이지 실전 - 2
일반적인 쇼핑몰 레이아웃 구조
global navigation bar(gnb)
local navigation bar(lnb)
local navigation bar 에 있는 리스트 들은 굳이 float left 속성을 주지 않아도 header li 속성에서 후손 선택자를 이용해 이미 float : left 를 통해 설정을 해놓았기 때문에
아무리 깊은 곳에 있는 li 태그라도 해당 속성이 적용되어 리스트의 구조가 수평으로 만들어지게 된다.
해당 강의에서 사용된 소스 코드를 다운 받아서 직접 분석해보자.

직접 특정 사이트의 레이아웃을 짜는 걸 연습해보면서 후에 phaser 를 이용한 웹 게임과 해당 웹 게임을 실행 시킬 수 있는 웹 사이트를 구축하는데
도움이 될 수 있을 것.
웹 사이트의 레이아웃을 만들어보더라도 Spring MVC 패턴을 이용하여 직접 사이트를 구축하는 방식으로 만들어보자.
웹 사이트 구축 이후, 자바 공부할 때 미리 받아놨던 Apache 서버와 연동하여 다른 컴퓨터에서도 해당 웹 사이트에 접속 할 수 있게 만들어보자.(유료 웹 호스팅과 같은 경로를 통해 직접 서비스를 해보자.
제작한 웹 게임을 서비스하기 위한 웹 사이트는 다른 게임 웹 사이트의 레이아웃을 참조하여 구성을 어떻게 가져갈지 생각해본다.
