서호준 웹 게임 개발 프로젝트

도시괴담인 SCP 재단의 각종 괴담 컨텐츠를 활용한 웹 게임을 제작한다. 건물형 scp 인 scp-1983, 통칭 어디로튼 통하는 문을 공략 및 탈출하는 공포 장르의 웹 게임이며 게임 제작 툴은 phaser, 게임 코딩 언어는 HTML5, Modern Javascript(ES2020) 을 사용한다. 게임 개발 이후 해당 게임을 실행시킬 수 있는 웹 사이트를 개설하여 직접 서비스를 해보는 것이 1차 목표이며 이를 활용하여 취업 시 프로젝트 포트폴리오로 활용하는 것이 2차 목표이다.
