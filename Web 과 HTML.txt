Web 이란?
사전적인 의미로 거미줄이라는 뜻으로 IT 에서는 WWW 라는 말의 줄임말로서 사용된다.
전 세계의 컴퓨터들을 연결하여 정보 공유를 하는데 주 목적이 있다. 그리고 이렇게 연결되어 있는 상태를 네트워크에 연결되어 있다고 한다.
(네트워크 상에서 WWW 서비스로 정보 공유를 한다.) 

웹 이란 1개 이상의 사이트가 연결되어 있는 인터넷 서비스의 한 가지 형태
인터넷이란, 1개 이상의 네트워크가 연결되어 있는 형태를 말한다.
프로토콜 : 네트워크 상에서 약속한 통신규약(Http, FTP, SMTP, POP, DHCP)
IP : 네트워크 상에서 컴퓨터를 식별할 수 있는 주소
DNS : IP 주소를 인간이 쉽게 외울수 있도록 매핑한 문자열
Port : IP 주소가 컴퓨터를 식별할 수 있게 해준다면, Port 번호는 해당 컴퓨터의 구동되고 있는 프로그램을 구분할 수 있는 번호이다.
http://www.sbaz.seoul.kr:80/kr/index (80은 기본 포트번호이기 때문에 생략해도 된다, 프로토콜 또한 마찬가지)
(http:// - 프로토콜, www.sbaz.seoul.kr - 컴퓨터 주소(DNS 를 통한 IP 주소로 변경), 80 - port 번호, kr/index - information path)

HTML 이란?
Hyper Text Markup Language 의 약자로서 웹 문서를 기술하는 언어
웹 문서를 표현하기 위해 태그들로 구성되어 있다.
Hyper(Hyper link) - html 문서의 가장 큰 특징 : 하이퍼링크를 통해 다른 페이지로 이동함
html 태그들을 하나씩 학습해 나간다.

html로 기본 골격을 만들어놓고 레이아웃, 색깔 등 디자인 요소를 가미해주는 작업을 css 작업이라고 한다.
css : Cascading Style Sheets 의 약자 - html 문서를 디자인적으로 예쁘게 만들어 정보 전달을 좀 더 효율적으로 하기 위해 만들어진 문서이다.
html 은 정보를 가지고 있고, css 는 화면상의 레이아웃 등을 컨트롤하는 문서라고 생각하면 된다.
메뉴배치, 폰트 색상 및 크기, 전체적인 레이아웃 등
