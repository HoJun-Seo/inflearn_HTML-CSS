스마트폰에서는 어떤 레이아웃이 어떻게 적용되는가?

viewport : 웹 브라우저의 html 문서에 특정한 정보를 나타내기 위한 태그로 meta 태그가 있다.
Pc 가 아닌 스마트폰에서 사용할 웹 브라우저에는 viewport 를 명시해줘야 한다.
meta 태그 : 내가 만들고자 하는 웹 문서에 문서만이 가지는 특정한 설정 정보들을 값으로 입력해주는 태그
모바일 웹 브라우저에서는 브라우저의 화면 설정과 관련된 정보를 제공하기 위한 방법으로 뷰 포트 meta 태그를 사용한다.
Pc 는 모니터에 따라 브라우저가 특별하게 달라보이진 않는다.
하지만 스마트폰 과 같은 모바일 기기는 제조사마다 엄첨 많은 사이즈 들이 존재한다.(일반 스마트폰, 태블릿 등등)
사이즈에 따라 좌우로 여백이 생기지 않고 단말기의 화면이 작던 크던 상관없이 항상 화면이 꽉 차게 나타난다.

meta 태그 예시
ex:
<meta charset="utf-8" name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no" />

뷰 포트 속성들
width : 화면의 너비 (device-width : 단말기의 가로 사이즈를 그대로 채택한다.)
height : 화면의 높이
initial-scale : 초기 확대 비율
user-scalable : 확대 및 축소 가능 여부
minimum-scale : 최소 비율
maximum-scale : 최대 비율
