전체와 특정태그의 혼합

선택자란?
특정 태그를 선택하여, 해당 태그의 속성을 변경하는 목적으로 사용된다.
글자 또는 배경의 색을 선택하여 선택한 태그의 속성을 변경한다.
css 언어는 html 문서를 기반으로 하여 해당 문서 안에 녹아들어 있는 문서이다.
css 언어가 작성되는 영역은 html 의 head 태그 안에 style 태그를 사용해줌으로서 해당 태그 안에 css 언어를 코딩해 나갈수 있다.
style 태그안에 중괄호를 사용하여 css 언어를 작성하는데 해당 중괄호의 이름으로서 들어가는 html 태그의 이름을 선택자라고 한다.

tag 선택자
특정 태그를 선택하여 css 속성을 적용할 수 있다.

* 선택자
전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택하는 의미가 있다.

전체와 특정 태그의 혼합
* 선택자로 전체를 선택한 후 속성을 변경하고 싶은 태그들을 선택자로 따로 지정하여 태그 별로 속성을 바꿔준다.

id(#) 와 class(.)
tag, id, class 혼합
속성 선택자

id(#) 와 class(.) - 태그의 id와 class 를 이용하여 css 에서 속성을 설정할 수 있다.
각 태그의 유일한 id 값 또는 class 값
id 값을 주고 나면 css 에서 속성을 줄때 div 태그를 선택자로 지정하는 것이 아니라 각 태그마다 지정해 놓은 id 값을 통해서
해당 태그의 속성을 변경시켜 줄 수 있다.
class 또한 id 와 비슷한 개념으로 볼 수 있다.
id 앞에는 #을 붙여서 css 로 코딩을 하고, class 앞에는 . 을 붙여서 css 로 코딩을 해준다.

속성 선택자
tag의 속성을 선택하여 css 의 속성을 설정 할 수 있다.

후손 및 자손 선택자
tag 의 속성을 선택하여 css의 속성을 설정할 수 있다.
자손 선택자? - 어떤 태그 바로 밑에 있는 태그를 선택할 경우 자손 선택자를 활용한다고 한다.
후손 선택자? - 어떤 태그의 밑에 있는 모든 태그들을 선택할 경우 후손 선택자를 활용한다고 한다.

동위 선택자
태그의 동등한 위치를 판단하여 css 속성을 설정 할 수 있다.
동위 선택자에는 두가지 타입이 있다, + 선택, ~ 선택
+ 선택 : 해당 태그의 아래에 있는 동등한 위치의 태그들 중에 지정한 태그를 찾는데, 아래에서 가장 첫번째로 만나는 지정된 태그를 선택한다.
~ 선택 : 해당 태그의 아래에 있는 동등한 위치의 태그들 중에 지정한 태그들을 모두 선택한다.
